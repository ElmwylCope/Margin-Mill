Margin Mill Pilot Landing Page — Standalone HTML
=================================================

Use this file:
- margin_mill_pilot_landing_standalone.html

This is the corrected version:
- CSS is embedded directly inside the HTML.
- JavaScript is embedded directly inside the HTML.
- No separate styles.css or script.js files are required.
- You can upload this one HTML file as your landing page.

Before publishing:
1. Rename margin_mill_pilot_landing_standalone.html to index.html.
2. Search for YOUR_EMAIL_HERE.
3. Replace YOUR_EMAIL_HERE with your preferred Margin Mill contact email.
4. Upload the renamed index.html to your static host.

This avoids the formatting issue caused by separated CSS/JS files not loading.
